# sgit
Git implemented in C#. The hash value is calculated to be the same as git.

## Implemented commands
- `sgit init`
- `sgit add {file}|{directory}`
- `sgit commit -m "{message}"`
- `sgit branch`
- `sgit branch {name}`
- `sgit branch -d {name}`
- `sgit status`
- `sgit log`
- `sgit log -n {num}`
- `sgit cat-file {hash}`
